Welcome to Sudoku! The objective of the game is to fill the
board such that each row, column, and box (as arranged by
color) with exactly one of the digits 1-9.

Disclaimer: the page may take some time to load when the
"Start game" or "New puzzle" buttons are clicked, since these
buttons cause new, unique random Sudoku puzzles to be
generated.

Here's a basic guide to the controls on this page:

  • To start a game, press the "Start game" button.
  • To obtain a new puzzle, click the "New puzzle" button.
  • Each game is timed; to pause a game's timer, press the
    "Pause game" button. To resume a game, press the
    "Resume game" button.
  • To reset a board, click the "Reset board" button; this will
    reset the board to the relevant blank puzzle and not
    restart the timer.
  • To reset the background colors of the board – which may
    be altered after checking the puzzle – click the "Reset
    colors" button.
  • To check a puzzle for correctness, click the "Check
    puzzle" button. The cells that were incorrect when the
    puzzle was checked will remain red until the next time the
    button is pressed, unless Live Check mode is in progress
    or the "Reset colors" button is clicked.
  • To solve a puzzle, click the "Solve puzzle" button. Doing
    so will terminate the game.
  • This game has two modes: Classic and Live Check. A
    Classic mode game is a regular Sudoku game. A Live
    Check game includes the feature of checking whether
    input values are correct. The mode with the yellow
    highlighted button is the mode currently being played in.
    Users may switch modes during games by clicking the
    button corresponding to the desired mode.
  • To get hints, enter "h"s into the desired spots and click
    the "Get hint" button. The first time the button is pressed
    during a run of the program, an alert button will provide
    these same instructions for emphasis.
